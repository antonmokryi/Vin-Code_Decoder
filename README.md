# VIN Decoder

## Опис
Простий веб‑додаток для декодування VIN. Є історія запитів і сторінка деталей.

## Демо
https://antonmokryi.github.io/VinCode-Decoder/

## Бібліотеки
- React
- Vite
- Zustand
- React Router
- Axios

## Локальний запуск
1. npm install
2. npm run dev
3. Відкрий адресу з терміналу (зазвичай http://localhost:5173/).
